Workflow

From a source library to finished teaching materials

WhatA clickable map of the education workflow. Each step has its own page.
WhyTeachers and learners need to understand the purpose of each step, not just the command to run.
NextStart with step 1, or use markdown-mentor start for a guided terminal run.

The workflow now has a clearer split. Markdown Library Maker creates the source library file. Markdown Mentor then uses that file for teaching and learning work.

01 You Collect sources Gather the files the learner will be taught from. 02 Markdown Library Maker Make the Markdown library file Convert the source folder into one AI-readable Markdown library. 03 Markdown Mentor Check it for teaching use Run a readiness check before asking AI to plan lessons. 04 You + AI Decide what to teach Create a Teaching Brief that sets the focus. 05 You Choose how to teach it Pick or adapt a Pedagogy Specification. 06 You Choose which materials to make Pick a Teaching Materials List for the purpose. 07 AI, checked by you Plan the materials Create and approve a Teaching Materials Inventory. 08 AI, checked by you Generate each material Draft one Markdown teaching material at a time. 09 Markdown Mentor Export Turn finished Markdown into practical formats.

Teaching note

The source library is not the lesson. It is the evidence base. The teaching steps add focus, level, pedagogy, practice, review, and export.
