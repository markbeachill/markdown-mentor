This page follows docs/canonical-workflow.md. Each step links to a page with the teaching view, what to do, and the technical command or prompt.

A. Create and develop your source materials library

  1. Create a Markdown Mentor project folder
  2. Collect your source materials in the source folder
  3. If needed, ask AI what other source materials would be useful
  4. Make your Markdown library file
  5. If needed, list or remove files from the Markdown library

B. Plan your teaching approach

  1. Create or customise your teaching approach file
  2. Check the teaching approach against the Markdown library with AI
  3. Choose and customise a teaching materials pack

C. Create your teaching materials

  1. Ask AI to create teaching materials as separate Markdown files
  2. Choose or customise your Markdown style file
  3. Process the teaching materials into styled output
