Workflow

From a folder of files to finished teaching materials

WhatA clickable map of the Markdown Mentor workflow. Each step has its own page.
WhyTeachers and learners need to understand the purpose of each step, not just the command to run.
NextStart with step 1, or use markdown-mentor start for a guided terminal run.

Markdown Mentor separates the work clearly. You collect sources and make teaching decisions. The software converts, checks, and exports. The AI proposes, plans, and drafts when you paste prompts into your own chatbot.

01 You Collect sources Gather the files the learner will be taught from. 02 Markdown Mentor Build the content pack Convert the source folder into one AI-readable Markdown pack. 03 Markdown Mentor Check the pack Run a readiness check before asking AI to plan lessons. 04 You + AI Decide what to teach Create a Teaching Brief that sets the focus. 05 You Choose how to teach it Pick or adapt a Pedagogy Specification. 06 You Choose which materials to make Pick a Teaching Materials List for the purpose. 07 AI, checked by you Plan the materials Create and approve a Teaching Materials Inventory. 08 AI, checked by you Generate each material Draft one Markdown teaching material at a time. 09 Markdown Mentor Export Turn finished Markdown into practical formats.

Shortcut

To be walked through every step in the terminal, run markdown-mentor start. The site pages explain what each step is for; the command helps you do the software parts.
