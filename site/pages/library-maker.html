Make Markdown Library

Turn source files, folders, ZIPs, and existing libraries into one structured Markdown file.

Make Markdown Library is the general-purpose source tool. It combines your source files into one Markdown library file that AI can read as a set of separate sources.

Normal-user command

python make-markdown-library.py make

This uses the default folders: source files from 1-source-files/, output to 2-markdown-library/markdown-library.md.

You can also give the source and destination folders explicitly:

python make-markdown-library.py make 1-source-files 2-markdown-library

This reads 1-source-files/ and writes:

2-markdown-library/markdown-library.md
2-markdown-library/markdown-library-manifest.md

What it can read

  • folders
  • individual files
  • ZIP files
  • nested ZIP files
  • existing Markdown library files

If the source folder contains an existing Markdown library file, the tool imports each source section from that library as a separate source. It does not add the old library as one large file.

Project prompt files

Files named prompt-*.md are ignored by the library builder. They are instructions for the user, not source material.

Duplicates

Duplicates are skipped by default. If a duplicate is skipped, the tool prints:

not added - filename

Use this only if you deliberately want duplicate source material:

python make-markdown-library.py make --allow-duplicates

Useful commands

python make-markdown-library.py make
python make-markdown-library.py add more-sources.zip
python make-markdown-library.py list
python make-markdown-library.py remove-file 3
python make-markdown-library.py check-file

Developer/tester package commands

make-markdown-library new 1-source-files -o 2-markdown-library/markdown-library.md
make-markdown-library add 2-markdown-library/markdown-library.md more-sources.zip
make-markdown-library list 2-markdown-library/markdown-library.md
make-markdown-library remove-file 2-markdown-library/markdown-library.md 3
make-markdown-library check-file 2-markdown-library/markdown-library.md

This tool checks file structure only. It does not decide whether sources are good enough for teaching.
