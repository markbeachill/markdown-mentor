Markdown Mentor tool

Guide the teaching workflow and export finished Markdown materials.

Home Workflow Start project Make Markdown Library Markdown Mentor Download Prompts Examples Style file Install

The normal-user teaching script is make-teaching-materials.py. It does not send files to AI. It reminds the user of the workflow and exports Markdown files saved in 5-draft-materials/.

Show the guide

python make-teaching-materials.py guide

Export

python make-teaching-materials.py export docx
python make-teaching-materials.py export pptx
python make-teaching-materials.py export html
python make-teaching-materials.py export pdf

The script reads the style file at style/style.md and saves finished files in 6-final-exports/.

Developer/tester package command

markdown-mentor guide my-project
markdown-mentor export 5-draft-materials -f docx -o 6-final-exports -s style/style.md

Teaching judgement still happens in AI prompts and user review. The Python tools handle files.
