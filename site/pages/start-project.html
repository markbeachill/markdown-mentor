Start a project

Create one working folder for the whole teaching-material project.

A Markdown Mentor project is one normal folder with the same structure every time.

Normal-user route

  1. Make a new folder for your project.
  2. Download setup-markdown-mentor.py into that folder.
  3. Open a command window in that folder.
  4. Run:
python setup-markdown-mentor.py

The setup file downloads make-markdown-library.py and make-teaching-materials.py, then creates the project folders.

Project structure

my-project/
  setup-markdown-mentor.py
  make-markdown-library.py
  make-teaching-materials.py
  README-FIRST.md
  1-source-files/
    prompt-find-more-source-materials.md
  2-markdown-library/
  3-teaching-approach/
    teaching-approach.md
    prompt-create-teaching-approach.md
    prompt-check-teaching-approach.md
  4-teaching-materials-pack/
    teaching-materials-pack.md
    prompt-create-teaching-materials.md
  5-draft-materials/
  6-final-exports/
  style/
    style.md

What to do next

  1. Put source files in 1-source-files/.
  2. Run python make-markdown-library.py make.
  3. Run python make-teaching-materials.py guide when you want the step-by-step reminder.

Developer/tester route

markdown-mentor new-project my-project
