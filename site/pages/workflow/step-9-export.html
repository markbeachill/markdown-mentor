Step 9

Export

Who leads this step: Markdown Mentor

Teaching and learning view

This step makes the materials usable in the real world: handouts for students, notes for teachers, slides for class, or web pages for sharing.

WhatMarkdown Mentor converts the finished Markdown files into DOCX, PPTX, HTML, or PDF.
WhyMarkdown is easy to edit, but most learners and teachers need familiar formats. Export keeps the writing source simple while still producing usable files.
NextOpen the exported files, check formatting, and make final edits before using them with learners.

What to do

  1. Put the approved Markdown files in one folder.
  2. Choose the export format you need.
  3. Use a style profile if you want a particular look.
  4. Open every exported file before sharing it.

Check before moving on

Ask: does the exported format help the learner use the material easily?

Technical appendix

This is the practical detail for running or finding the step in the repo.

Commands

markdown-mentor export materials -f docx -o exports markdown-mentor export materials -f pptx -o exports markdown-mentor export materials -f html -o exports markdown-mentor export materials -f pdf -o exports -s library/style-profiles/student-handout.json

Files and details

  • DOCX is best for handouts, teacher notes, worksheets, and answer keys.
  • PPTX is best for slide overviews and class presentation material.
  • HTML is best for web pages or quick sharing.
  • PDF export uses LibreOffice if it is installed. If it is not available, export to DOCX or HTML and use “Save as PDF”.
  • Style profiles live in library/style-profiles, and the website Style page can make new ones.
