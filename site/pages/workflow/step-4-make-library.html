Step 4. Make your Markdown library file

Make your Markdown library file

Teaching and learning view: Combine the source files into one AI-readable Markdown library.

What to do

Run this command:

python make-markdown-library.py make

This uses the default folders: source files from 1-source-files/, output to 2-markdown-library/markdown-library.md.

You can also give the source and destination folders explicitly:

python make-markdown-library.py make 1-source-files 2-markdown-library

The tool also creates 2-markdown-library/markdown-library-manifest.md and handles folders, files, ZIPs, nested ZIPs, and existing Markdown library files.

If the source folder contains an existing Markdown library, the tool imports the sources inside that library as separate source sections. It does not add the old library as one large file, so AI can still read the new library as a set of separate files.

Duplicates are skipped by default. If a duplicate is skipped, the tool prints not added - filename. To include duplicates deliberately, add --allow-duplicates.

Check before moving on

Make sure the file or folder named above exists and says what you need it to say. The user reviews AI work before continuing.
