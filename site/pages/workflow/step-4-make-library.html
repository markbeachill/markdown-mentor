Step 4. Make your Markdown library file

Make your Markdown library file

Home Workflow Start project Make Markdown Library Markdown Mentor

Teaching and learning view: Combine the source files into one AI-readable Markdown library.

What to do

Run this command:

make-markdown-library new 1-source-files -o 2-markdown-library/markdown-library.md

The tool also creates 2-markdown-library/markdown-library-manifest.md and handles folders, files, ZIPs, and nested ZIPs.

Check before moving on

Make sure the file or folder named above exists and says what you need it to say. The user reviews AI work before continuing.
