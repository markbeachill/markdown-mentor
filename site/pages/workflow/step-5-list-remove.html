Step 5. List or remove files from the Markdown library

List or remove files from the Markdown library

Teaching and learning view: Check what went into the library and remove accidental or duplicate sources.

What to do

List, then remove if needed:

python make-markdown-library.py list
python make-markdown-library.py remove-file 3

The number comes from the list command. The tool updates the library and manifest and makes a backup.

Duplicates are skipped automatically when you make or add to a library, unless you use --allow-duplicates. If the tool skips a duplicate, it prints not added - filename.

Check before moving on

Make sure the file or folder named above exists and says what you need it to say. The user reviews AI work before continuing.
