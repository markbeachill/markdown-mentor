Step 6

Choose which materials to make

Who leads this step: You

Teaching and learning view

This step turns a teaching aim into a useful set of learner-facing and teacher-facing outputs. A revision pack, tutor session, and practical lesson need different materials.

WhatChoose a Teaching Materials List from the library. It names the kinds of documents you want, such as a handout, slide overview, quiz, answer key, or homework task.
WhyChoosing the set before generation keeps the work focused. It also helps the AI plan a coherent pack instead of producing random documents.
NextUse the chosen list with the material specifications to plan the exact inventory.

What to do

  1. Choose the list that matches the teaching situation.
  2. Remove any material type you do not need.
  3. Add a material type only if the source pack can support it.
  4. Keep the first run small, then expand later if the plan works.

Check before moving on

Ask: would this set of materials help the learner move from explanation to practice to review?

Technical appendix

This is the practical detail for running or finding the step in the repo.

Files and details

  • Material lists live in library/materials-lists.
  • Current examples include getting started, revision pack, tutor session, and practical lesson.
  • The shape of each material type is defined separately in library/material-specs/teaching-material-specifications.md.
  • The list goes into the inventory prompt.
