Workflow / Step 8

Step 8

Generate each material

Who leads this step: AI, checked by you

Teaching and learning view

This step creates the materials learners and teachers will actually use. Generating one item at a time makes quality control easier and keeps the work editable.

WhatFor each inventory item, ask the AI to write one material in Markdown. Read it, edit it, and save it as a separate .md file.
WhyLarge AI outputs are harder to inspect. Separate files make it easier to check accuracy, improve tone, regenerate one weak item, and export different formats.
NextWhen the Markdown files are ready, export them to Word, PowerPoint, HTML, or PDF.

What to do

  1. Generate only one inventory item per AI request.
  2. Check factual accuracy against the Markdown library/content pack.
  3. Check that the material follows the pedagogy and material specification.
  4. Save each approved result in a folder such as materials.

Check before moving on

Ask: could a learner or teacher use this material without extra explanation from me?

Technical appendix

This is the practical detail for running or finding the step in the repo.

Files and details

  • Useful prompt: Generate one teaching material.
  • Use # for the title, ## for sections, and simple lists and tables where needed.
  • Do not add HTML to the generated Markdown unless you have a specific reason.
  • End each material with a short “Sources used” section so it remains inspectable.
