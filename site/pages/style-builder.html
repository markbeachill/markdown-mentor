Create a Markdown style file

Use style/style.md to control exported document appearance.

The style file controls how exported documents look. It is Markdown, so users can read and edit it like the other project files.

The project style file lives here:

style/style.md

Example style file

# Style

## General style

- body_font: Calibri
- body_size_pt: 11
- heading_font: Calibri
- heading_color: 1F3864

## Word document style

- h1_size_pt: 20
- h2_size_pt: 16
- h3_size_pt: 13

## Slide style

- title_size_pt: 32
- body_size_pt: 20

## HTML style

- html_title: Teaching material

## PDF style

- use_docx_conversion: true

How it works

The export tool applies the settings it understands. If a setting is unclear or unsupported, it uses the default.
