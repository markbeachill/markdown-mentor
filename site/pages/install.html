Install for testing

Install the early prototype and create a test project.

Home Workflow Start project Make Markdown Library Markdown Mentor Prompts Examples Style file Install

Markdown Mentor is still an early prototype. These instructions are for testers who are comfortable installing a Python tool.

Check Python

python --version

You need Python 3.10 or newer. On Mac or Linux, try python3 --version if python does not work.

Install from the repository folder

pip install -e ".[dev]"

Create a project

markdown-mentor new-project my-project

Show the guide

markdown-mentor guide my-project

First project command

cd my-project
make-markdown-library new 1-source-files -o 2-markdown-library/markdown-library.md
