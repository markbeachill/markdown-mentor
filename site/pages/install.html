Install and setup

Use the simple download route, or install the full package for development.

Most users: download one setup file

You do not need to clone the repository.

Download setup-markdown-mentor.py

python setup-markdown-mentor.py

This downloads the two working scripts and creates the project folders in the current folder.

Check Python

python --version

You need Python 3.10 or newer. On Mac or Linux, try python3 --version if python does not work.

Optional source conversion helper

Plain text, Markdown, HTML, and CSV can be read directly by the simple script. Word, PDF, and PowerPoint files may need MarkItDown.

python -m pip install "markitdown[all]"

Optional export helpers

python -m pip install python-docx python-pptx

PDF export needs LibreOffice.

Developers and testers

Use this only if you are working on the repository itself.

pip install -e ".[dev]"
pytest
