Install

Two things to install

WhatMarkdown Mentor needs Python, a free tool called MarkItDown, and then Markdown Mentor itself.
WhyMarkItDown reads your source files. Without it, no content pack can be built. There is no fallback: this step is required.
NextFollow the steps below in order. Each says what to type and what you should see.

Step 1: Check you have Python

MarkItDown needs Python 3.10 or newer. Python is a free programming language the tool runs on. Open a terminal (on Windows, search for "Command Prompt"; on Mac, open "Terminal"; on Linux, open your terminal app), then type:

$ python --version

If you see version 3.10 or higher, go to Step 2. If not, install Python from python.org/downloads. On Windows, tick "Add Python to PATH" during install. If python does not work, try python3 everywhere on this page.

Step 2: Install MarkItDown

MarkItDown is a free tool from Microsoft that turns Word, PDF, PowerPoint, and other files into Markdown. Install it with:

$ pip install "markitdown[all]"

This may take a minute. If pip is not found, try python -m pip install "markitdown[all]".

Step 3: Install Markdown Mentor

Download or clone this project, then run this from inside the markdown-mentor folder (the one with the README in it):

$ pip install -e .

Step 4: Check it worked

$ markdown-mentor --version

If you see a version number, you are ready. Go to the Workflow page and run the small example, or run markdown-mentor start to be walked through it.

Optional: PDF export

To export straight to PDF, install LibreOffice, a free office suite. Then use -f pdf when you export. If you would rather not install it, export to Word or web and use "Save as PDF" in your own software. PDF is the only feature that needs LibreOffice.

If something goes wrong

If a command is "not recognised", Python may not be on your PATH; reinstall it and tick "Add Python to PATH" on Windows. If you get a permissions error, add --user to the install command. If MarkItDown installs but Markdown Mentor still says it is missing, you may have more than one Python; use the same one for both.

Official MarkItDown pages

MarkItDown is made by Microsoft, not by this project. For updates and help with MarkItDown itself, see github.com/microsoft/markitdown and pypi.org/project/markitdown.
