Markdown Mentor

Make a source library, use AI to draft teaching materials, then export them.

Workflow Start project Make Markdown Library Markdown Mentor Prompts Examples Style file Install

1. Make a Markdown library

Put your source files in one project folder. Use make-markdown-library to combine files, folders, ZIPs, and nested ZIPs into one Markdown library file.

Read about Make Markdown Library

2. Plan with AI

Use the prompts inside the project folder to create a teaching approach and a teaching materials pack. The AI helps; the user decides.

See the workflow

3. Export finished files

Save AI-created Markdown materials in 5-draft-materials/, then use markdown-mentor export to make DOCX, PPTX, HTML, or PDF files.

Read about Markdown Mentor

Main rule: Python checks files. AI and the user check teaching fit.
