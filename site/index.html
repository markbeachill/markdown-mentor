A workflow for teachers, tutors, and learners

Turn your teaching files into teaching materials.

Markdown Mentor does the parts a computer is good at: it gathers your files, checks them, and exports finished materials. You and an AI chatbot do the thinking.

What this is

WhatA step-by-step way to turn a folder of source documents into worksheets, slides, handouts, and quizzes.
WhyTeaching files are scattered across many formats. Handing them to an AI without structure gives broad, inconsistent results. This adds the structure.
NextRead the workflow below, then install the tool and run the small example.

Markdown is a simple text format used for headings, lists, and links. Markdown Mentor uses it because it is readable, portable, and easy to turn into other formats. The tool itself never talks to an AI. Instead, it gives you ready-made prompts to paste into any AI chatbot.

The workflow

  1. You
    Collect sources. Gather the files the learner will be taught from.
  2. Markdown Mentor
    Build the content pack. Convert the source folder into one AI-readable Markdown pack.
  3. Markdown Mentor
    Check the pack. Run a readiness check before asking AI to plan lessons.
  4. You + AI
    Decide what to teach. Create a Teaching Brief that sets the focus.
  5. You
    Choose how to teach it. Pick or adapt a Pedagogy Specification.
  6. You
    Choose which materials to make. Pick a Teaching Materials List for the purpose.
  7. AI, checked by you
    Plan the materials. Create and approve a Teaching Materials Inventory.
  8. AI, checked by you
    Generate each material. Draft one Markdown teaching material at a time.
  9. Markdown Mentor
    Export. Turn finished Markdown into practical formats.

Honest about the tool

Worth knowing

The readiness check is a simple, rule-based helper, not an AI judgement. AI drafts can be wrong, so always read and edit what you get. PDF export uses LibreOffice if you have it; if not, export to Word or web and use "Save as PDF".

The aim is a tool that is clear, honest, and easy to act on. It keeps you in control. The software converts, organises, checks, and exports. The AI proposes, plans, and drafts. You select, edit, approve, and teach.
