Two plain tools for AI-ready Markdown work

Make a clean Markdown library. Then build teaching materials from it.

Markdown Library Maker gathers source files into one structured Markdown file. Markdown Mentor uses that file to help you plan, draft, and export teaching materials.

Tool 1

Markdown Library Maker

Creates one structured Markdown file from a file, folder, or ZIP. It can also add more files later.

  • general-purpose
  • useful beyond education
  • keeps source boundaries and fingerprints

Tool 2

Markdown Mentor

Uses a Markdown library file to help plan, draft, check, and export teaching materials.

  • education-focused
  • uses prompts and human review
  • exports Markdown to DOCX, PPTX, HTML, and PDF

The teaching workflow

  1. You
    Collect sources. Gather the files the learner will be taught from.
  2. Markdown Library Maker
    Make the Markdown library file. Convert the source folder into one AI-readable library.
  3. Markdown Mentor
    Check it for teaching use. Run a readiness check before asking AI to plan lessons.
  4. You + AI
    Decide what to teach. Create a Teaching Brief that sets the focus.
  5. You
    Choose how to teach it. Pick or adapt a Pedagogy Specification.
  6. You
    Choose which materials to make. Pick a Teaching Materials List for the purpose.
  7. AI, checked by you
    Plan the materials. Create and approve a Teaching Materials Inventory.
  8. AI, checked by you
    Generate each material. Draft one Markdown teaching material at a time.
  9. Markdown Mentor
    Export. Turn finished Markdown into practical formats.

Honest about the tool

Worth knowing

This is still an early local prototype. Ordinary users should not have to clone a repo or use commands in the long term. For now, the command-line version is mainly for testers and technical users.

The aim is a tool that is clear, honest, and easy to act on. The software converts, organises, checks, and exports. The AI proposes, plans, and drafts. You select, edit, approve, and teach.
