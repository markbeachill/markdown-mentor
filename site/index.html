1. Set up a project folder

The setup file downloads the two working scripts and creates the standard folders for your sources, prompts, drafts, style file, and exports.

Start a project

2. Make a Markdown library

Put your source files in 1-source-files/. Run python make-markdown-library.py make to create one source library for AI.

Read about Make Markdown Library

3. Plan and create with AI

Use the prompt files in the project folders to create a teaching approach and draft materials as Markdown files.

See the workflow

4. Export finished files

Save Markdown materials in 5-draft-materials/, then run python make-teaching-materials.py export docx.

Read about exports

Main rule: Python checks files. AI and the user check teaching fit.
