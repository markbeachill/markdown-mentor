# Teaching Materials List: Practical Lesson

A Teaching Materials List is a chosen set of material types for one purpose.
This "Practical Lesson" list is for a hands-on lesson where learners do
something, not just read about it.

Materials to create:

- A lesson plan (aims, timings, and steps for the teacher).
- An activity sheet (what the learner does, step by step).
- A slide deck (to introduce and guide the activity).
- A worked example (a finished example to aim for).
- A reflection task (what did the learner learn, and what next).

Why these five: a practical lesson needs a plan, clear instructions, a model to
copy, and a moment to reflect, so the doing turns into learning.
