# Teaching Materials List: Getting Started

A Teaching Materials List is a chosen set of material types for one purpose.
This "Getting Started" list is a small, friendly first pack. Use it when you
want to introduce a topic.

Materials to create:

- A short slide overview (introduce the topic).
- A glossary (key terms in plain English).
- A student worksheet (explanation, worked example, simple practice).
- A question-and-answer set (a few easy questions with answers).

Why these four: together they let a learner meet a topic, learn its words,
practise it, and check understanding, without being overwhelmed.
