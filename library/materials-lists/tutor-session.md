# Teaching Materials List: Tutor Session

A Teaching Materials List is a chosen set of material types for one purpose.
This "Tutor Session" list is for a one-to-one or small-group tutoring session
built around a learner's needs.

Materials to create:

- Diagnostic questions (find out what the learner already knows).
- Worked examples (model the skill clearly).
- Targeted practice (tasks aimed at the learner's gap).
- A homework task (short practice to do before the next session).

Why these four: a good tutor session checks where the learner is, models the
skill, practises the exact gap, and sets focused follow-up work.
