# Prompt: Generate one teaching material

Use this prompt to create a single item from your approved inventory. Generate
materials one at a time. This keeps each one easy to read, check, and redo if
needed.

What the AI receives: your content pack, your Pedagogy Specification, the
matching Teaching Material Specification, and the one inventory item you want to
make now.

What the AI must produce: one finished teaching material, written in Markdown,
ready to save as a .md file and export with Markdown Mentor.

What you should check afterwards: that the material is accurate, matches the
brief and pedagogy, and only uses content from the sources. Edit the Markdown
before you export it if you wish.

---

You are writing one teaching material. Write only this one item.

You will receive:
1. A content pack (the source material).
2. A Pedagogy Specification (how to teach).
3. One Teaching Material Specification (the shape this material must take).
4. One inventory item (the exact material to create now).

Write the material in Markdown. Follow the Teaching Material Specification
exactly. Match the tone and approach in the Pedagogy Specification. Use only
information found in the content pack. If something needed is missing from the
sources, say so in a short note at the end rather than inventing it.

Formatting rules for the Markdown:
- Use "#" for the title, "##" for sections, "###" for sub-sections.
- Use "-" for bullet points and "1." for numbered lists.
- Keep lines short and plain.
- Do not add HTML.

At the very end, add a short section called "Sources used" listing which
sources from the pack you drew on.
